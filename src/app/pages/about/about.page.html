World Data

This app fetches JSON metadata from several REST endpoints from REST Countries API, that has information on every country in the world.

All countries are initially listed in alphabetical order. User can list countries by world region. Clicking on a list item routes to a country detail page with links to more detailed country info, a map and 'add to favourites'.

The Google Maps API is used to display the country map. The usual Google Maps functions are available, like zoom etc.
